## Como executar o projeto

1. Clone do repositório :

```

$ git clone https://github.com/Dev-Messias/todo-list.git

```
> Acesse a página utilizando `cd todo-list`

2. Instale as dependências :

```

$ npm install

```

3. Inicie o projeto :


```

$ npm run dev

```
